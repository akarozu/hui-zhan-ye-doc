## 1. 业务目标

解决客户经理在慧展业工作台中**快速查找管户客户、筛选目标客户、查看客户详情**的业务需求，支撑管户效率提升和客户KPI（AUM/绩效/新增客户数）达成。

---

## 2. 功能范围

### ✅ 做

| 模块 | 功能说明 |
|------|----------|
| **客户列表** | 展示当前客户经理的管户客户，滚屏加载（无限滚动，不加分页） |
| **多条件组合筛选** | 5个筛选项：产品类型、风险等级、客户分类、最近活跃时间、资产区间；筛选条件变更即时联动刷新列表，无需确认按钮 |
| **客户详情页** | 新窗口打开，5Tab结构：基本信息 / 持有产品 / 资产概况 / 互动记录 / 投资偏好 |

### ❌ 不做

- 客户批量操作（导出、群发短信/微信）
- 客户编辑、分配、转移
- 潜客名单（仅限管户客户）
- 跨客户经理数据查看

---

## 3. 用户故事

| ID | 角色 | 故事 | 收益 |
|----|------|------|------|
| US01 | 客户经理 | 我想看到所有管户客户列表，以便掌握整体管户规模 | 减少页面切换，快速获取全量客户 |
| US02 | 客户经理 | 我想按产品/资产/活跃度等多条件筛选，以便快速定位目标客户 | 提升管户效率，节省查找时间 |
| US03 | 客户经理 | 我想查看客户详情页，以便了解客户全貌后开展营销动作 | 减少跨系统跳转，一个页面掌握客户全貌 |

## 4. 页面规格

### 4.1 客户列表页

**路径**：`/customer/list`

**布局**：

```
┌──────────────────────────────────────────────────────────────┐
│  顶栏：「管户客户」标题                                       │
├──────────────────────────────────────────────────────────────┤
│  搜索框（姓名/手机号）                                         │
├──────────────────────────────────────────────────────────────┤
│  [客户类型▲] [AUM▲] [优先级▲]              [🔍筛选按钮]     │
├──────────────────────────────────────────────────────────────┤
│  [活跃中] [私行客户]  ← 已选中的筛选项tag（可关闭）            │
├──────────────────────────────────────────────────────────────┤
│  列表区域（滚屏加载）                                          │
│  ┌────────────────────────────────────────────────────────┐  │
│  │ 张三  │ 138****8888 │ [创富客户] │ AUM: 50万  │ →│  │  │
│  │  ▌(优先级彩条)                                     │  │
│  └────────────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────────────┘
```

> 注：筛选按钮点击后从底部弹出筛选面板（Bottom Sheet），内含产品类型/优先级/客户分类/资产区间/活跃时间五个筛选项组。「最近活跃」字段不再在列表行展示。

**筛选项规格**：

| 筛选项 | 类型 | 选项值 | 交互 |
|--------|------|--------|------|
| 产品类型 | 下拉多选 | 理财产品 / 存款 / 基金 | 选择即联动 |
| 优先级 | 下拉多选 | 🔥 紧急 / ⚠️ 优先 / ○ 正常 | 选择即联动 |
| 客户分类 | 下拉多选 | 私行客户 / 财富客户 / 创富客户 / 有效客户 / 大众客户 / 潜力客户 | 选择即联动 |
| 资产区间 | 自定义输入 | 最低-最高（万），实时生效 | 输入即联动 |
| 活跃时间 | 下拉单选 | 近7天 / 近30天 / 近90天 / 90天以上 | 选择即联动 |

> ⚠️ **资产区间已从预设按钮改为自定义输入**：筛选项调整为「最低」和「最高」两个数字输入框，单位为万，实时生效，无需确认按钮。

**列表 Chip 排序**（三chip横排，右端筛选按钮）：

| Chip | 说明 | 排序逻辑 |
|------|------|---------|
| 客户类型 | 按类型A→Z / Z→A | 点击切换升序/降序 |
| AUM | 按资产高低 | 点击切换升序/降序 |
| 优先级 | 紧急 > 优先 > 正常 | 点击切换升序/降序 |

**列表字段**：

| 字段 | 说明 | 格式 |
|------|------|------|
| 客户姓名 | — | 文本，脱敏规则同现有页面 |
| 手机号 | — | 前3后4，中间 `****` |
| 客户分类 | 蓝色标签，取自客户分类字段 | tag-blue |
| 优先级 | 卡片左侧彩色条：红色=紧急/橙色=优先/灰色=正常 | 左侧3px彩条 |
| AUM | 管户总资产 | 数字+万，如「50万」，保留一位小数 |

**列表排序**：默认按**AUM降序**，支持点击列头切换排序字段（待确认）

**滚屏加载规则**：
- 首屏加载20条
- 滚动至底部自动加载下一页
- 加载中显示骨架屏/loading态
- 全部加载完成显示「已全部加载，共X条」

### 4.2 客户详情页

**路径**：`/customer/detail/:id`（新窗口打开）

**Tab结构**：

| Tab | 内容 | 数据来源 |
|-----|------|---------|
| 基本信息 | 姓名/手机/身份证/风险等级/客户分类/管户经理/开户时间 | 客户信息系统 |
| 持有产品 | 基金/理财/存款/国债，按产品类型分组展示，显示产品名称/份额/市值 | 持仓系统 |
| 资产概况 | 总AUM/持仓结构饼图/历史持仓趋势（近6个月） | 持仓系统+报表 |
| 互动记录 | 营销/维护历史，时间轴倒序显示，包含拨打/拜访/产品推荐记录 | CRM系统 |
| 投资偏好 | 风险测评结果/偏好板块/持仓习惯 | 客户画像系统 |

**互动记录规格**：
- 显示最近12条记录
- 每条记录包含：时间、互动类型（拨打/拜访/推荐）、互动内容摘要
- 按时间倒序排列

**⚠️ 待确认项**：
- 「最近活跃」的定义：最后登录/最后交易/最后联系？
- 互动记录是否需要展示具体沟通内容（脱敏规则）？

## 5. 验收标准

| ID | 功能 | 验收条件 |
|----|------|----------|
| AC01 | 列表加载 | 进入页面默认加载管户客户，首屏≤1s |
| AC02 | 滚屏加载 | 滚动至底部自动加载下一页，无分页控件，加载中显示loading态 |
| AC03 | 多条件筛选 | 任意筛选条件变更后列表实时刷新（≤500ms），无手动确认按钮 |
| AC04 | 跳转详情 | 点击客户行，新窗口打开详情页 |
| AC05 | 详情Tab切换 | 5个Tab切换正常，切换不刷新页面，切换≤200ms |
| AC06 | 互动记录显示 | 显示最近12条营销/维护历史，按时间倒序 |
| AC07 | 数据权限 | 客户经理只能看到自己管户的客户，数据隔离有效 |
